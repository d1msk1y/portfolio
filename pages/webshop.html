Eldas Webshop

An integrated into a Business-Software product catalog

Webshop is an integrated into a Business-Software product catalogue that lists products imported into the Mongo DB from an XML/CSV file.

The Webshop interacts with the Business-Software's API to add desired items to an invoice/project/ticket with all the required params like: Buy/Sell Price, Manufacturer, Quantity and Description.

The backend is on Node.JS with handlebars for lighweight server-side rendering. The Webshop's REST API provides endpoints for manual import; UI import with progress bar and import status; and the actual catalogue page.
